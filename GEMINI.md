# Gemini CLI Project Instructions

This file contains project-specific instructions and context for the Gemini CLI agent.

## Tech Stack
- **Backend:** Python, FastAPI, Supabase, LangChain/Agents
- **Frontend:** React, TypeScript, Vite, Tailwind CSS

## Commands
- **Backend Setup:** `pip install -r requirements.txt`
- **Frontend Setup:** `cd frontend && npm install`

## Conventions
- Follow PEP 8 for Python code.
- Use TypeScript for all frontend code.
- Ensure all new features are tested.
